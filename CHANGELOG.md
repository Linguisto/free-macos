## [0.1.0](https://github.com/Linguisto/free-macos/releases/tag/v0.1.0) - 2026-01-15

### Added

- Initial release of `free` for macOS.
- GNU-compatible memory and swap usage output.
- Support for human-readable output (`-h`).
- Support for specific units: bytes (`-b`), kibibytes (`-k`), mebibytes (`-m`), and gibibytes (`-g`).
- Continuous monitoring with delay (`-s`) and count (`-c`) options.
- Totals display (`-t`).
- macOS-specific memory breakdown details (`--details`).
- Basic `README.md` with installation and usage instructions.
